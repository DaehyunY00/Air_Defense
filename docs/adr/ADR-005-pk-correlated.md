# ADR-005 — 재교전 요격확률 상관(pkCorrelated) (Phase 5, ⑨)

## 맥락
종전 모델은 매 발사의 pk를 **독립 추출**한다 → 같은 표적을 n회 재교전하면 누적 격추확률이 `1 − Π(1−pk_i)`로 항상 상승한다. 그러나 재교전 실패는 흔히 **체계적**이다: 불리한 교전기하·ECM·표적 특성(저고도·저RCS 무인기)은 발사 사이에 공유돼, 첫 발이 빗나간 이유가 다음 발에도 남는다. **2022.12.26 소형 무인기 5대 남파 시 반복 대응에도 전량 미격추**가 대표 사례 — 독립 모델은 이런 "재교전이 구제하지 못하는" 실패를 과소평가하고 재교전 효과를 과대평가한다.

## 선택지
- **A. 공유 잠재(shared frailty)**: 표적별 잠재값 `frailty`를 최초 교전에서 1회 추출, 발사별 비교 균일난수를 `u = ρ·frailty + (1−ρ)·신규추출`로 혼합. ρ=0 독립, ρ=1 완전상관.
- **B. 결정론적 감쇠**: `pk_eff = pk·decay^(tries−1)`. 평균은 낮추나 발사 결과는 여전히 독립(상관 미도입) — "전량 실패"의 상관 구조를 설명 못 함.
- **C. 표적 hardness 계층모형**: 위협유형별 상관 파라미터를 문서 추정 — 데이터 부재로 근거 등급 하락.

## 결정
**A**(`features.pkCorrelated`, **기본 OFF**). ρ는 `PK_CORR_RHO=0.7`(문서 기본), `features.pkCorrelation` 숫자로 재정의 가능(스윕용). `_onEngageEnd`에서 OFF면 legacy(`raw()<pk`), ON이면 표적별 `_frailty`(최초 교전 지연추출) + 발사별 신규추출을 ρ 혼합해 `u<pk` 판정.

**되돌리기/그리기 수:** OFF → 발사당 `raw()` 1회(legacy와 완전 동일, 스윕 검증 불일치 0). ON → 표적당 `_frailty` 1회 추가(발사당 raw는 그대로 1회). 동역학 지문·스냅샷은 **기본 OFF라 불변**.

## 근거
- **기본 OFF인 이유(조건 2)**: ρ 값의 실측 근거가 없다(근거 등급 **C**). 그리고 결론을 유의미하게 움직인다(아래) → "C등급 & 유의미 이동 → 기본 OFF" 규칙에 정확히 해당.
- **A를 택한 이유**: B(감쇠)는 평균만 낮추고 발사 결과의 **양의 상관**을 만들지 못해 "5대 전량 실패=우연이 아닌 공통원인"을 재현 못 한다. A는 frailty 공유로 `P(전량 미격추)`를 독립 예측보다 높여, 사례와 방향이 일치한다.
- **혼합 모형의 한계(정직)**: `u`의 주변분포는 엄밀한 균일이 아니다(평균 0.5 유지·단조). 단일 발사 격추율 ≈ pk는 근사로만 성립하며, 효과는 주로 **재교전 이득 축소**로 나타난다. 이는 개념 모형으로서 수용하고 vv-report 한계에 명시.

## 결론 영향(스윕, `scripts/phase5-sweep.mjs`, To-Be seed1~20 × SC1/2/3 × x1/1.5/2.5)
| ρ | 전체 격추율 | 독립 대비 |
|---|---|---|
| OFF(독립) | 50.01% | — |
| 0.0(ON) | 49.94% | −0.06p (≈독립, 되돌리기 근사) |
| 0.3 | 48.36% | −1.64p |
| 0.5 | 45.89% | −4.12p |
| **0.7(기본)** | **44.27%** | **−5.73p** |
| 0.9 | 40.72% | −9.28p |
| 1.0 | 40.15% | −9.86p |

- 2022.12.26 앵커(SC1 x2.5, uav 비중 높은 시나리오 대리): 독립 79.2% → ρ=0.7 74.1% → ρ=1 65.0%. **재교전 이득이 상관과 함께 단조 축소** — 사례 방향과 일치. (엄밀 위협유형별 격추율은 Phase 7 신규지표로 분리 계측 예정 — 현재는 시나리오 전체 대리, 한계 명시.)
- **기본 설정(OFF)에서 결론(To-Be 개선폭)은 불변 → Phase 5는 기본 에스컬레이션 0%.** ON(ρ=0.7)로 켜면 격추율 −5.73p로 As-Is·To-Be가 함께 하락(재교전 이득은 To-Be가 더 크므로 To-Be가 더 많이 깎임 = 방어 성과의 **낙관 편향 제거**). 이는 옵션이며 기본 결론에 미반영.

## 되돌리는 법
- 플래그: `features.pkCorrelated=false`(기본) → legacy 독립 추출. 지문·스냅샷 불변.
- ρ 조정: `features.pkCorrelation`(0~1)로 스윕. 원복하려면 `_onEngageEnd`의 상관 분기·`PK_CORR_RHO`·`_frailty` 필드 제거.

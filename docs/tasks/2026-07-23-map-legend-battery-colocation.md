# 지도 범례 토글·공동 포대 중첩 시각화

## 구현 결과

- 우측 하단 범례를 기본 열린 `<details>`로 바꿔 제목을 눌러 열고 닫을 수 있게 했다.
- legacy 10개 세트의 ECS·MFR·사수 좌표를 해당 포대의 동일 개념 위·경도로 통일했다.
- 6개 MINI/FULL 배치에서도 포대 전속 MFR/Patriot radar가 포대·ECS와 동일 위·경도를
  사용하도록 정규화했다. 센서 안테나 고도와 출처 메타는 유지한다.
- Leaflet과 SVG fallback 모두 동일 좌표 자산을 한 사이트 마커에 기능별 기호로 겹쳐
  표시한다. 배지는 구성 자산 수를, 팝업은 모든 구성 자산의 역할·범위·부하를 보여준다.
- 자산 요약에 전체 활성 노드 수와 지도 사이트 수, 중첩 사이트 수를 함께 표시한다.
- 라벨은 지도 hit target 밖에 그려 인접 사이트 클릭 영역을 불필요하게 넓히지 않는다.

- Leaflet 및 SVG fallback 중첩 마커·범위 링·연결선 토글 회귀 통과.
- 로컬 브라우저에서 범례 열기/닫기, legacy 10개 중첩 사이트, MINI 8개 중첩 사이트,
  배치 전환과 자산 요약을 확인했으며 콘솔 오류는 없었다.
- `git diff --check`: 통과.
